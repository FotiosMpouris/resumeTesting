Hard-E v3.0

Production Deployment — Voice-First AI Sales Agent for Contracting

A voice-first, multi-tenant AI platform built for the contracting industry. Hard-E is an intelligent sales orchestrator — managing customer relationships, navigating CRM data in real time, and holding lifelike conversations with a configurable personality. Not a chatbot. An AI teammate.

Highlights

  • Sub-100ms time-to-first-audio with sentence-level streaming (Cartesia Sonic 3)
  • Multi-tenant from day zero — each company gets isolated data, memory, and personality
  • Three-tier memory: session, long-term summaries, and Redis-backed knowledge tier
  • Audio-reactive 56-bar frequency visualizer responding to AI voice in real time
  • Safety tiers — green (read), yellow/red (write actions require confirmation)
  • 5-step onboarding questionnaire activates each tenant instance

Tech Stack

  • Claude Sonnet (orchestration brain) · Cartesia Sonic 3 (voice) · OpenAI TTS (fallback)
  • Next.js · TypeScript · Redis (memory) · Web Audio API · WebRTC (roadmap)
  • AWS · Docker · Nginx · production SSL

OpenCare

Production · Live Daily · Free · The MVP for the Poor People App

A care coordination platform I built because I needed it. As the primary caregiver for my parents, I was managing doctor appointments, insurance renewals, home repairs, and grocery runs across siblings in different states — through group texts and memory. OpenCare replaced that chaos with organized care circles, assignable tasks, automated reminders, and email summaries that keep everyone informed without forcing anyone to download an app.

It's live. It's free. It's being used right now to coordinate care for real people. OpenCare is the first operational MVP under the Poor People App umbrella — proof that the larger PPA vision is already shipping working software.

Live Features

  • Care Circles — private workspaces for family, community groups, or care teams
  • Task Management — full lifecycle: medical, insurance, household, financial, legal, transportation, personal care, social
  • Recurring Tasks — daily, weekly, monthly, quarterly, yearly, or custom intervals (every 5 years for document renewals)
  • Offline Contacts — assign tasks to people who don't use the app; they get email notifications
  • Daily Telegram & Email Digests — automated morning summary of all tasks due today
  • Sentinel Identity Verification — government ID + selfie via Persona (same tech banks use)
  • AI Agent REST API — full programmatic control with API key auth, currently driven by TheGreekClawd
  • Google Calendar Integration — one tap to add any task to your calendar
  • Progressive Web App — installable from any browser, no App Store required

Tech Stack

  • Next.js 14 · TypeScript · Tailwind CSS · React
  • Clerk Auth (enterprise SSO) · Supabase / PostgreSQL with Row Level Security
  • Persona (government ID verification) · Resend (transactional email) · Telegram Bot API
  • AWS EC2 · Docker · Nginx · Let's Encrypt SSL
  • Custom REST API with Bearer token auth, multi-tenant scoping, safety guardrails

TheGreekClawd

Active Build · Personal Digital Employee · Not Publicly Accessible

An autonomous AI agent built as a full digital employee — not an assistant that answers questions, but an operator that ships code, manages workflows, and runs independently between sessions. Built on the open-source OpenClaw framework, communicating through Telegram, controlling Claude Code to execute development work end-to-end. Currently integrated with the OpenCare REST API to manage care tasks via natural language commands.

This entry exists on the website not because you can visit it — you can't, it's personal infrastructure — but to mark where this practice currently sits on the agentic curve. Digital employees are no longer a concept. They are a current operational reality.

What It Does Today

  • Telegram-native interface — already operational, coordinating in real time
  • Claude Code integration — autonomous coding agent control and execution
  • OpenCare API integration — creates, updates, and monitors care tasks via natural language
  • Built on open-source OpenClaw agentic framework
  • End-to-end task execution — receives a goal, ships the result

Why It's Listed Here

Building digital employees on top of open-source agentic frameworks is the frontier work happening right now. This is a working showcase of where development is headed — autonomous agents that don't just generate code, but operate as colleagues.

Matrix Rain Studio

Standalone Art Piece — Interactive Generative Visual

An interactive Matrix Rain art generator. Not a passive screensaver — a personalization studio where users can type their own text into the rain, switch between Katakana, Code, Greek, Binary and Hex character sets, customize colors, speed, density, glow, and trail length, then download their composition as a wallpaper or share it via URL.

Features

  • Type any text and watch your own words rain down
  • 6 character sets — Katakana, Code, Greek, Binary, Hex, Custom
  • 7 color presets plus full custom color picker
  • Real-time controls — speed, density, glow, trail length
  • Bright lead character & random bold drop options
  • Download canvas as PNG wallpaper
  • Share creations via URL-encoded settings
  • Fullscreen mode + keyboard shortcuts (S, F, D, R, Esc)
  • Mobile-responsive with slide-up control sheet
  • HiDPI-aware, zero dependencies, ~25 KB total

Tech Stack

  • HTML5 Canvas + Vanilla JavaScript (no frameworks, no backend)
  • requestAnimationFrame render loop with device-pixel-ratio scaling
  • URLSearchParams for shareable state encoding
  • GitHub Pages hosting with custom subdomain (matrix.fotiosmpouris.com)
