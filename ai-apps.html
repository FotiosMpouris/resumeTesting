Hard-E v3.0 — Conversational AI Sales Agent

A voice-first, multi-tenant AI platform for the contracting industry. Not a chatbot — an AI teammate that manages customer relationships, navigates CRM data in real time, and holds lifelike conversations with a configurable personality. Sub-100ms audio. Tenant-isolated from day zero.

  • Sub-100ms TTS — Cartesia Sonic 3 with sentence-level streaming; audio plays while the AI is still thinking. OpenAI TTS fallback.
  • 3-tier memory — session (24hr TTL), long-term conversation summaries, Redis-backed knowledge tier for company facts and learned patterns
  • Safety tiers — green (reads execute freely), yellow/red (writes require confirmation or explicit readback)
  • Audio-reactive UI — 56-bar frequency visualizer (Web Audio API) reacts to AI voice in real time

OpenCare — Care Coordination with an AI Agent API

Production · Live Daily · Free · The MVP for the Poor People App

A live care coordination platform built because I needed it — coordinating elder care for my parents across siblings, caseworkers, and community helpers. The differentiator on this page: the platform exposes a full REST API for AI agents. TheGreekClawd already uses it to create, assign, and monitor real care tasks via natural language. Multi-tenant by design — any user can generate API keys and connect their own agent.

  • AI Agent API — Bearer auth, SHA-256 key hashing, per-user scoping, safety guardrails (no delete endpoints, explicit confirmation for cancellations)
  • Care Circles & Offline Contacts — assign tasks to people who don't have the app; they get email notifications with full details
  • Sentinel ID Verification — government ID + selfie via Persona (same platform used by banks)
  • Daily Telegram & email digests — automated morning summaries; PWA installable from any browser

TheGreekClawd — My First Digital Employee

Active Build — Based on OpenClaw Open Source Framework

TheGreekClawd is an autonomous AI agent built to function as a full digital employee — capable of executing real development work, coordinating tasks through Telegram, and controlling coding agents via Claude Code. Based on the open-source OpenClaw framework, this represents the frontier of agentic AI: not just an assistant that answers questions, but an agent that ships code, manages workflows, and operates independently between sessions. Already communicating through Telegram and operational.

  • Telegram-native interface — operational, coordinating in real time
  • Claude Code integration — autonomous coding agent control and execution
  • OpenCare API integration — manages care tasks via natural language commands
  • End-to-end task execution — receives a goal, ships the result

PPA Mobilization Agent — AI Campaign Engine

The first live operational agent in the Poor People App's AI workforce. GPT-4o with a tightly defined writer's persona — Orwell's clarity, Washington's resolve, Odysseus' strategy — generating content grounded in a live knowledge base, deployed to Telegram and Nostr with one click. Self-scoring after every generation; learning memory injects past performance into future sessions.

  • 5 content formats — text posts, Sora video scripts (with TTS audio), GIF prompts, DMs, email outreach
  • Self-improving — agent scores its own knowledge base usage (0–10) and specificity, stores results in DynamoDB
  • One-click broadcast — Telegram bot + 3 Nostr relays; user feedback loop flags exemplary outputs
  • Banned-language rules — emails reject "revolution," "game-changer," and pitch language by spec
