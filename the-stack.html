The Stack.

Tools, systems, and the honest version.

Background

I came into tech from the side door — years of running a restaurant, client work, creative projects, and eventually teaching myself to build. Art school, animation, game dev in Unity, then Python, then AI hit and everything accelerated.

Most of what I know, I learned by building things that needed to work. Hard-E started as a favor for a friend's business. OpenCare started because my family needed it. The Grillin Greek is ten years of feeding people. These aren't portfolio pieces — they're things I use or things that serve real people.

I'm comfortable in the deep end of AI agent architecture, but I'm also still learning every day. The stack below is what I'm working in right now. It changes fast.

What I'm Building Right Now

  • Hard-E v3.0 — Voice-first AI sales agent. Claude Sonnet, Cartesia Sonic, AWS EC2. Multi-tenant, 3-tier memory, sub-100ms audio. Built for a real contracting business.
  • OpenCare — Care coordination platform. Next.js, Supabase, Clerk, full REST API. Live, free, used daily to coordinate elder care in my own family.
  • All Angles Exterior — AI-enriched lead pipeline for exterior remodeling. GPT-4o vision for house photo analysis, Perplexity for property research, DynamoDB, SES. I'm the technical lead.
  • TheGreekClawd — Autonomous dev agent on OpenClaw. Controls Claude Code via Telegram to execute tasks between check-ins. Still experimental.
  • FotiFoti Art Agent — Serverless pipeline on AWS Lambda. Picks art, colorizes it, writes captions, posts to Instagram and Facebook autonomously.
  • PPA Mobilization Agent — GPT-4o content engine with self-scoring memory. Broadcasts to Telegram and Nostr.

Infrastructure & Tools

  • Languages — Python, JavaScript/TypeScript, C#
  • Frameworks — Next.js, React, Tailwind, Unity

How I Work

  • Build-first — I prototype early and ship often. Most of these systems went from idea to production in weeks, not months.
  • AI-native — I develop inside multi-agent environments daily. Cursor, Claude Code, and autonomous agents are part of the workflow, not afterthoughts.
  • Full lifecycle — From database schema to deployment pipeline to brand identity. I don't hand off — I own the whole thing.
  • Real users — Everything I build is either used by paying clients, my own family, or an active community. No demos, no mockups.
